#
# Record는 사진과 위치를 함께 기록하는 다이어리입니다.

### 사용자는 자신의 여행에 대한 다이어리를 생성하고 포스트에 사진과 설명을 작성합니다. 사진의 GPS정보나 직접 입력한 위치 정보를 기반으로 사진,설명과 함께 지도에 사진의 위치에대한 핀을 지정하고 보여줍니다.
